# Overview
This package is a set of 25+ components and 450+ icons from [Palantir's Blueprint Design System](https://blueprintjs.com/docs/). 
Blueprint is Palantir's react-based UI toolkit for the web.

## Supported Components

### Components
- Button
- Callout
- Card
- Icon
- MenuItem
- MenuDivider
- ProgressBar
- Spinner
- Tag
- Toast
- TreeNode

### Form Controls
- Switch
- Radio
- Checkbox
- Slider
- RangeSlider
- RadioGroup

### Form Inputs
- FileInput
- TextArea
- Input

### Typography
- Blockquote
- Code
- Headings
